# Convert a workflow object to a data frame

This method converts a \`workflow\` object into a data frame summarizing
its steps.

## Usage

``` r
# S3 method for class 'workflow'
as.data.frame(x, ...)
```

## Arguments

- x:

  A \`workflow\` object.

- ...:

  Additional arguments (not used).

## Value

A data frame summarizing the workflow steps.

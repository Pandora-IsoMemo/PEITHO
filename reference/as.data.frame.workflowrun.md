# Convert a workflowrun object to a data frame

This method converts a \`workflowrun\` object into a data frame
summarizing its state.

## Usage

``` r
# S3 method for class 'workflowrun'
as.data.frame(x, ...)
```

## Arguments

- x:

  A \`workflowrun\` object.

- ...:

  Additional arguments (not used).

## Value

A data frame summarizing the workflow run state.

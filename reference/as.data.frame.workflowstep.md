# Convert a workflowstep to a data frame

This method converts a \`workflowstep\` object into a data frame format,
which can be useful for tabular display or exporting.

## Usage

``` r
# S3 method for class 'workflowstep'
as.data.frame(x, ...)
```

## Arguments

- x:

  A \`workflowstep\` object.

- ...:

  Additional arguments (not used).

## Value

A data frame with one row representing the workflow step.
